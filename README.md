# Recursive component

## Git clone

```
git clone https://github.com/andreyskorchenko/recursivecomponent.git
```

## Change directory

```
cd recursivecomponent
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```
